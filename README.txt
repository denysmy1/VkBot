1. Для работы бота вам необходимо иметь группу, от лица которой будет работать бот.
1) Создаем группу в вк.
2) Получаем токен группы. Для этого на страницы группы мы заходим в настройки - работа с API - нажимем "создать ключ"
3) Настройка API группы. В разделе "работа с API заходим во вкладку Long Poll API. Включаем Long Poll AP и ставим версию 5.80

2. Работа с программой
1) Открываем файл main.py удобым вам текстовым редактором или интерпритатором Python. 
2) В переменную self.group_id = прописываем айди своей группы без ковычек и решетки, просто цифрами.
в self.api_token = "Ваш токен" вместое текста "Ваш токен" вставляем токен который вы получили, в кавычках, как в коде.
Так же в файле server_manager.py меняем строку #айди группы на айди вашей группы, без ковычек и решетки. 

3. Запуск бота
1) Чтобы запустить бота необходимо установить все необходимые библиотеки.
Для этого нам необходим установленный Python3.
В терминале поочередно приписываем команды:
pip install vk_api
pip install requests
pip install bs4
2) После установки всех необходимых библиотек можно включать бота, запустив файл server_manager.py

4) Установка на сервер
1) После того как вы установили бота на сервер, для его запуска в терминал нужно перейти в папку с ботом, ипрописать команду:
python3 server_manager.py > bot.log 2>&1 &
